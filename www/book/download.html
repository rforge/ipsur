Download Options

You can get a PDF copy (in color) of the published edition of IPSUR from CRAN, and you can get a PDF of the development edition from R-Forge; see the the Installation page for details. Note that the Sweave sources of both of those documents are stored alongside their PDF siblings.

If you would like to print your own copy of the published edition of IPSUR then you may want to use the following publication-ready PDF of the book. It has been typeset in black-and-white to optimize the code display. All of the fonts have been embedded so your printer should not have trouble with any of the symbols.

Publisher Quality PDF
IPSUR.pdf

If you would like a transparent copy of the published edition of IPSUR, then download the following zip file. It contains all of the LaTeX source code and all of the figures (in .pdf and .eps). These can be used (in principle) to generate your own personal copy of the published edition of IPSUR, and you are FREE to modify and distribute your own version of it under the constraints of the GNU-FDL.

LaTeX source and Images
IPSUR.zip

Finally, you can download the LyX source file, IPSUR.lyx, which was used by the author to write IPSUR. It is essentially the Sweave source code for the published edition, but LyX conceals the intermediate step of generating the LaTeX source file and figures. This one text file will generate a PDF of the entire published edition of IPSUR with a single click. Enjoy. (I certainly have.)

Master LyX File
IPSUR.lyx

You will need the BibTeX file to generate the bibliography.

You are free to print your own copy of the development edition of IPSUR, but think twice. For one thing, the electronic copy is in color which does not show up well on black-and-white paper. For another, the electronic copy is optimized for web viewing which means that not all of the fonts are embedded in the PDF - your printer may not render all of the symbols correctly. Moreover, unless you have a very fancy printer you will be printing only on one side of the page - which wastes paper.

If you would like to print your own copy of the IPSUR development edition then the best approach would be to download the Sweave source, compile your own PDF in black-and-white, and be sure to embed all of the fonts. Here is a script that I use to generate IPSUR for publication.
